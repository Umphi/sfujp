# Практические работы Java (Юркевич И.А. ЗКИ25-18Б. Вариант 16)

## Практическая работа 1 
"Программная реализация простых алгоритмов с использованием языка Java"

Цель работы: Ознакомиться с синтаксисом языка Java и его основными конструкциями.

Общая постановка задачи: Необходимо реализовать "консольное" приложение, которое имеет примитивный текстовый интерфейс и меню, состоящее как минимум из четырех пунктов:

Ввод исходных данных, как вручную, так и сгенерированных случайным образом (выбор предоставляется пользователю)
- Выполнение алгоритма по заданию
- Вывод результата
- Завершение работы программы.

Входные данные: текст. Результат работы алгоритма: количество строчных и прописных букв каждого из алфавитов (не менее двух - английский и русский), а также число знаков препинания и пробелов.
