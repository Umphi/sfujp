# Практическая работа 2 (Юркевич И.А. ЗКИ25-18Б. Вариант 16)


## "Создание классов и объектов в языке Java"

Цель работы: Ознакомиться с возможностями языка Java по созданию классов и объектов.

Общая постановка задачи: Необходимо реализовать класс согласно варианту "Постройка", в том числе придумать поля (атрибуты) для вашего варианта: числовые (целое и вещественное, не менее двух) и текстовые (не менее двух). Все поля класса должны быть закрытыми, необходимо реализовать методы доступа и модификации. В классе должны быть реализованы конструктор по умолчанию (без параметров) и конструктор с параметрами.

Необходимо предложить и реализовать метод, который будет возвращать значение, основанное на имеющихся полях. Например, для автомобиля, на основе его массы и количества пассажирских мест можно возвращать класс необходимых прав B, C, D.

Главная функция программы содержит массив объектов (допускается использование ArrayList).

Необходимо реализовать приложение, которое имеет простой текстовый интерфейс и меню, состоящее как минимум из шести пунктов:
- Добавление пустого объекта к массиву (используется конструктор по умолчанию).
- Добавление объекта с данными, заполненными пользователем (используется конструктор с параметрами).
- Редактирование любого поля (по выбору пользователя) любого объекта (по индексу).
- Вывод в консоль информации обо всех объектах, включая результат самостоятельно придуманной функции из общей постановки задачи.
- Сортировка массива (списка) объектов по любому из полей (по возрастанию).
- Завершение работы программы
